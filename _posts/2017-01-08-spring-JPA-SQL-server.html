the sequence to generate a spring jpa with sql server is like the following:

1. pom.xml with sqljdbc4.jar

2. specify sql properties in prop files

3. Entity class (with @Id)

4. Interface repo implemnts crudrepo and jparepo

5. EmployeeMgr (@Autowired repo)

6. MainConfig (which specify the Mgr bean)

6. You can use it...
